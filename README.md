# L8teMod / L8teBot

Ein Twitch Lurk Bot und Web-Dashboard.

## Installation

### Via Docker (Empfohlen)

Das Repository erstellt automatisch Docker Images, die über die GitHub Container Registry (GHCR) verfügbar sind.

Siehe [DOCKER_INSTALL.md](DOCKER_INSTALL.md) für Details zur Installation ohne manuelles Bauen.

### Entwicklung / Lokales Bauen

Um den Bot lokal zu entwickeln oder selbst zu bauen, siehe [INSTRUCTIONS.txt](INSTRUCTIONS.txt).

## Einrichtung & Keys

Damit der Bot läuft, brauchst du API Keys von Twitch und Google.
Eine genaue Anleitung findest du hier: **[SETUP_GUIDE.md](SETUP_GUIDE.md)**

## Features

- Twitch Chat Integration
- Web Dashboard zur Konfiguration
- Automatische Nutzer-Erfassung (Lurk-Funktion)
- Gemini AI Integration für Chat-Antworten
